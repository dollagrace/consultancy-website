# Dolla Business Consultants Website Demo

A responsive business consultancy website demo inspired by the M&A Registrars website structure.

## Project Overview

This project is a modern frontend website built using:

* HTML5
* CSS3
* JavaScript

The website includes:

* Responsive navigation bar
* Hero section with call-to-action
* Services section
* About / Why Choose Us section
* Testimonials
* Contact form
* Scroll animations
* Mobile responsiveness

## Purpose

This project serves as:

* A demo concept for a potential client website
* A frontend learning project
* A portfolio project

## Features

* Responsive design for desktop and mobile
* Smooth hover transitions
* Scroll-based animations
* Corporate UI with emerald color palette

## Future Improvements

* Backend contact form integration
* WhatsApp direct chat
* CMS/blog support
* SEO optimization
